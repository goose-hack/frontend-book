# Frontend Bookとは

「Frontend Book」は、プログラミング未経験からwebフロントエンドの案件を獲得するための学習ロードマップです。

webプログラミングを勉強したいけど、何を、どのサイトで勉強すればいいか分からない。という方に向けて、必要な技術と、そのおすすめの学習リソースをまとめました。

## 求人への応募に必要な技術

`React 未経験 求人`などでGoogle検索してヒットする求人を見てみてください。

おそらく、多くの求人で必須スキル・歓迎スキルに上がってくるのは以下のようなものになると思います。

- HTML, CSS
- JavaScript, TypeScript
- React.js, Next.js
- Git, GitHub

上記の技術を使用してポートフォリオを作れば、未経験可の求人に応募できるレベルになります。
