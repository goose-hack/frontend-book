# 導入 : Frontend Bookについて

「Frontend Book」は、Webフロントエンドエンジニアを目指す方向けの学習サイトです。

プログラミング未経験者を対象にしており、前提知識はPCの基本操作程度です。

実践的な技術でWebアプリを作り、アルバイトなどの実務で十分に働けるスキルを身につけることが、当サイトの目標です。

一緒に成長し、新しい可能性に挑戦していきましょう。

## フロントエンドエンジニアの仕事

フロントエンドエンジニアは、デザイナーと協力し、ユーザが直感的に操作できるように、ウェブサイトの外観と動作を実装する役割を果たします。

また、バックエンドエンジニアと連携して、必要なデータを適切なタイミングで取得・加工・表示するための設計と実装も担当します。

#### 個人で新しいサービスを作れる

個人でも一定のサービスを独自に開発することができ、コストを掛けずに新しい事業をスタートできます。

### デメリット

Web技術は変化が激しく、最新の技術に追従する必要があります。

新しい技術に触れることを楽しむ人に向いています。

## Frontend Bookで学ぶこと

フロントエンドエンジニアとして市場評価を高めるには、

- 実務経験
- ポートフォリオ (制作物)

の2つが重要です。

まずは十分なポートフォリオを作成し、早くアルバイトや実務経験を積むことを目指しましょう。

### 第一章 : Webサイトを作る

まずは基本的な技術であるHTML, CSS, JavaScriptを使ってWebサイトを作ってみましょう。

### 第二章 : 実務で使われる技術を使う

実務では第一章で扱ったHTML, CSS, JavaScriptを直接書く機会は少ないです。

この章では、React, tailwindcss, TypeScriptなどの実践的な技術を使用してWebサイトを構築します。

### 第三章 : チーム開発に必要なツールを使う

実務では複数のエンジニアが協力して開発するため、ソースコードの管理と共有にGit, GitHubを使用します。

この章では、その基本的な使い方を学びます。
