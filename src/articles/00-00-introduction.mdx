# Frontend Bookとは

「Frontend Book」は、プログラミング未経験からフロントエンドエンジニアの求人に応募するための無料学習サイトです。

PCの基本操作程度の前提知識を想定しています。

実践的な技術でWebアプリを作り切り、未経験可の求人に十分応募可能なスキルを身につけることが当サイトの目標です。

一緒に成長し、新しい可能性に挑戦していきましょう。

## 求人に応募するために必要なもの

エンジニアの採用では、**実務経験**と**ポートフォリオ**の2つが重要です。

ポートフォリオとは、スキルを示すための成果物のことで、自分で開発したアプリやコードを提出します。

エンジニアとして市場評価を高めていくための最初のステップは、目標とする案件に必要な技術を使用したポートフォリオを作り切り、早く実務経験を積むことです。

Frontend Bookでは、webフロントエンドの案件応募に必要な技術に絞ってポートフォリオを作成します。

<Link href="./getting-start">

まずは基本的な技術であるHTML, CSS, JavaScriptを使ってWebサイトを作ってみましょう。

このChapterでは、React, tailwindcss, TypeScriptなどの実践的な技術を使用してWebサイトを構築します。

**Chapter 3 : チーム開発に必要なツールを使う** (準備中)

実務では複数のエンジニアが協力して開発するため、ソースコードの管理と共有にGit, GitHubを使用します。

このChapterでは、その基本的な使い方を学びます。
