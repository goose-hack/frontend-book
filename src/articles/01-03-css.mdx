# 03 CSSでwebページにスタイルをつけてブラウザで表示する
## 目標

- CSSを用いてwebページにスタイルをつけることができる
- 作成したwebページをブラウザで表示できる

## CSSとは

## 準備

1. VSCodeでwebフォルダを開く
1. `New Files`から新たなファイルを作成し、`style.css`と名前を入力する

## セレクタの種類

## CSSで使用できるプロパティ

### 背景色

    ```
    body {backgroud-color: blue}
    ```

bodyに背景色を当ててみましょう

### フォントの色

    ```
    h1 {color: green}
    ```

h1に色をつけてみましょう

### 余白 padding

```
. {padding: 16px}
```
