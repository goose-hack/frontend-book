# 開発環境構築 : VScodeのインストール

Webページを作成するには、HTMLやCSS、JavaScriptなどのコードを記述する必要があります。

```html
<body>
  <h1>HTMLのイメージ</h1>
  <p>このようなコードを書く</p>
</body>
```

この作業には専用のテキストエディターが不可欠です。

一般的なテキストエディターであるメモ帳アプリでは、コードの書き込みが難しく、エラーや不具合の発見が複雑になります。

Visual Studio Code（VScode）は多くの開発者に愛用されている専用のテキストエディターです。

このチャプターではVScodeをインストールし、開発を始める準備を整えます。

## Visual Studio Codeの入手

<TabMdx>
  <div data-label="windows">
    1. [こちら](https://go.microsoft.com/fwlink/?LinkID=534107)からVScodeのインストーラーをダウンロードします。
    2. ダウンロードされたインストーラー **VSCodeUserSetup-\{version\}.exe** をダブルクリックして実行します。これには1分ほどしかかかりません。
  </div>
  <div data-label="mac">
    1. [こちら](https://go.microsoft.com/fwlink/?LinkID=534106)からVScodeをダウンロードします。
    2. Finderを開き、**option⌥ command⌘ L**を同時に押すとダウンロードフォルダが開きます。
    3. ダウンロードされたzipファイルをダブルクリックして解凍すると、**Visual Studio Code.app**が表示されます。
    4. Finderを別ウィンドウでもう一つ開き、**shift⇧ command⌘ A**を同時に押すとアプリケーションフォルダが開きます。
    3. **Visual Studio Code.app**をアプリケーションフォルダにドラッグします。
    4. アプリケーションフォルダからVScodeをダブルクリックして開きます。
    5. Dock内にあるアイコンを右クリックして、コンテキストメニューを表示し、「オプション」→「Dockに残す」を選択して、VScodeをDockに追加します。
  </div>
</TabMdx>

## 初期設定

インストールが完了すると、VScodeが起動します。初回起動時には、いくつかの初期設定が求められることがあります。言語やテーマの選択など、個人の好みに合わせて設定してください。

これで、VScodeのインストールと基本的なセットアップが完了しました。次のステップに進んで、実際にコーディングを始めましょう！

Happy coding!
