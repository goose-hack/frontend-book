# 開発環境構築 : VisualStudioCodeのインストール

Webページを作成するには、HTMLやCSS、JavaScriptなどのコードを記述する必要があります。

```html
<body>
  <h1>HTMLのイメージ</h1>
  <p>このようなコードを書く</p>
</body>
```

この作業には専用のテキストエディターが不可欠です。

一般的なテキストエディターであるメモ帳アプリでは、コードの書き込みが難しく、エラーや不具合の発見が複雑になります。

VisualStudioCodeは多くの開発者に愛用されている専用のテキストエディターです。

このチャプターではVisualStudioCodeをインストールし、開発を始める準備を整えます。

## VisualStudioCodeのインストール手順

<TabMdx>
  <div data-label="windows">
    1. [こちら](https://go.microsoft.com/fwlink/?LinkID=534107)からインストーラーをダウンロードします。
    2. ダウンロードされたインストーラー **VSCodeUserSetup-\{version\}.exe** をダブルクリックして実行します。これには1分ほどしかかかりません。
  </div>
  <div data-label="mac">
    1. [こちら](https://go.microsoft.com/fwlink/?LinkID=534106)からVisualStudioCodeをダウンロードします。
    2. Finderを開き、 **command⌘ + option⌥ + L** を同時に押すとダウンロードフォルダが開きます。
    3. ダウンロードされたzipファイルをダブルクリックして解凍すると、 **Visual Studio Code.app** が表示されます。
       ![screenshot](/img/01-01-mac-01.png)
    4. **command⌘ + N** でFinderを別ウィンドウで開き、 **command⌘ + shift⇧ + A** でアプリケーションフォルダが開きます。
    3. **Visual Studio Code.app** をアプリケーションフォルダにドラッグします。
       ![screenshot](/img/01-01-mac-02.png)
    4. アプリケーションフォルダから **Visual Studio Code.app** をダブルクリックして開きます。
    5. Dock内にあるアイコンを右クリックして、コンテキストメニューを表示し、「オプション」→「Dockに残す」を選択して、VisualStudioCodeをDockに追加します。
       ![screenshot](/img/01-01-mac-03.png)
  </div>
</TabMdx>

おめでとうございます! これでwebページを作る準備が出来ました。
