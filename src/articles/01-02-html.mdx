# HTMLでwebページ作ってブラウザで表示する

## 目標

画像のようなwebページを作成し、テキスト、リンク、リスト、画像など、基本的なHTMLの書き方を習得しましょう。

<Image src="/img/01-02-html.png" alt="html logo" width={512} height={350} />

## HTMLとは

HTML （HyperText Markup Language、ハイパーテキスト・マークアップ・ランゲージ）は、ウェブサイトのコンテンツの構造を作るために使うコードです。

```html
<body>
  <h1>HTMLのイメージ</h1>
  <p>このようなコードを書く</p>
</body>
```

このコードが何をしているのが見てみましょう。

```html
<!DOCTYPE html>
```

`<!DOCTYPE html>`は、この文書がHTML5という種類のHTMLドキュメントであることをブラウザに伝えるための宣言です。

```html
<html lang="ja">

</html>
```

これは、HTMLドキュメント全体を囲むタグです。`lang="ja"`は、このページが日本語で書かれていることを示します。

```html
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width" />
  <title>Hello World</title>
</head>
```

これは、ウェブページに表示されない情報を記述するためのタグです。

- `meta charset="utf-8"` : このページの文字エンコーディングがUTF-8であることを示します。これは、日本語などの多バイト文字を正しく表示するために必要です。
- `meta name="viewport" content="width=device-width"` : 様々なデバイスでこのページを適切に表示するために、ビューポートの設定を指示します。具体的には、デバイスの幅に合わせてページの幅を調整することを意味します。
- `<title>Hello World</title>` : このページのタイトルを定義します。これは、ブラウザのタブやブックマークなどに表示されます。
