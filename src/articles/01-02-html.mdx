### 準備

### HTMLで使用できるタグ

#### 段落

```
<p>アノマロカリスの生態</p>
```

- 文章の段落をつくりたいときに使う

#### グループ化

```
<div></div>
```

- 複数のタグにまとめてスタイルをつけるときにグループ化するために使う

#### 画像

```
<img src="/Hallucigenia.png" alt="可愛いハルキゲニアちゃん" />
```

- 画像を表示させたいときに使う
- srcには表示させたい画像の場所
- altには画像の簡単な説明

#### リンク

```
<a href="https://www.Sacabambaspis.jp">ホームページ</a>
```

- リンクを表示したいときに使う
- hrefにはリンクのURL
- リンクにしたい文字を`<a></a>`要素で囲む

#### 見出し

```
<h1>メインタイトル</h1>
<h2>最上位の見出し</h2>
```

- 見出しをして表示したい部分に使う
- `<h1>-<h6>`まであり、レベルで使い分ける
